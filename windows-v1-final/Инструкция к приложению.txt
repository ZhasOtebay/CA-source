Инструкция
!!!Обратите внимание, что приложение работает только с  форматом csv.  Для того чтобы изменить ваш excel файл положите его в \converter\csvconvert, 
запцустите  \converter\converter.exе измененный файл будет сохранен в \converter\source если структура таблицы подходит для расчетов(подробно в \converter\Инструкция к конвертеру.txt)

1. Распаковать архив в любую папку, проверьте чтобы не сработал антивирус

2. В папке windows-v1-final  запускаем  component-analyze.exe

3. Использование интерфейса
Главное окно: После запуска приложения откроется главное окно с заголовком "Анализ главных компонент".

Загрузка CSV файла:

Нажмите кнопку "Загрузить CSV файл".
Выберите файл в формате CSV, который содержит данные для анализа.  Убедитесь, что данные в файле соответствуют количеству колонок, которые вы собираетесь анализировать.
Если вы конвертировали файл то файл выберите из \converter\source. 

Ввод количества колонок:
После выбора файла появится запрос на ввод количества колонок. Нужно указывать сколько столбцов содержит ваша таблица для пересчета. Введите нужное количество колонок, содержащих данные для анализа, и нажмите ОК.

Анализ данных:

Приложение проведет анализ данных, включая нормализацию, расчет корреляционной матрицы, главных компонент и визуализацию.
Будут созданы файлы изображений: один с матрицей корреляции и другой с облаком точек главных компонент.
Сохранение результатов:

Результаты анализа будут сохранены в Excel-файл в папке output с именем, содержащим текущую дату и время.
Если папка output не существует, создайте её вручную, или добавьте в код создание папки автоматически.

Выход из приложения:

Чтобы закрыть приложение, нажмите кнопку "Выход".
Примечания
Убедитесь, что данные в CSV-файле правильно отформатированы. Неправильные данные могут привести к ошибкам в процессе анализа.
Если возникнут какие-либо ошибки, будет показано сообщение с описанием проблемы. Обратите внимание на детали, чтобы понять, что пошло не так.
